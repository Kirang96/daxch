# Compliance and Risk Controls

## Regulatory Positioning

- Daxch is a trading operations assistant, not a stock recommendation engine.
- User must explicitly input a ticker before analysis.
- AI output is framed as informational analysis and execution support.

## Mandatory Product Controls

1. Display disclaimer in onboarding, stock setup, and each AI output.
2. Record immutable audit logs for:
   - Input market snapshot
   - Portfolio context
   - AI prompt + response metadata
   - Decision and execution status
3. Keep user override controls:
   - Per-decision confirmation
   - Agent pause/stop
   - Emergency stop-all agents
4. Keep consent and terms acceptance history.

## Standard Disclaimer

This platform provides AI-generated analysis for informational purposes only and does not provide investment advice. You are solely responsible for all investment and trading decisions.
